# FUTURE_PE_01
Professional website copy and responsive website prototype for MK Cafe, Warangal using AI Prompt Engineering, HTML, and CSS.
MK Cafe Website

Project Overview:
MK Cafe Website is a simple and responsive website designed for a local cafe. The website provides information about the cafe, its services, menu highlights, and contact details.

Objective:
The objective of this project is to create professional website content for a local business using AI-powered prompt engineering and basic web development technologies.

Features:
->Responsive Homepage
->Services Section
->Call-to-Action (CTA) Sections
->Contact Information
->User-Friendly Design
->Mobile-Friendly Layout

Technologies Used:
->HTML5
->CSS3
->ChatGPT (for content generation)
